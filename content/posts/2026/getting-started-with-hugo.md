Welcome to my blog! This is the first post, and it's about the very setup you're reading right now.

## Why Hugo?

Hugo is one of the fastest static site generators available. It compiles an entire website in milliseconds, which means:

- **Fast builds** in CI/CD pipelines
- **No runtime dependencies** — just static HTML, CSS, and JavaScript
- **Great ecosystem** of themes and community support

## Why GitHub Pages?

GitHub Pages offers free, reliable hosting with:

- Automatic HTTPS via Let's Encrypt
- Custom domain support
- Integration with GitHub Actions for automated deploys

## Setting It Up

The basic workflow is:

1. Write posts in Markdown under `content/posts/`
2. Push to the `main` branch
3. GitHub Actions builds the site and deploys it automatically

Every post you write shows up on the homepage grouped by **year and month**, so readers can easily find recent content.
